# Exercícios C# - POO, Estruturas e Conceitos

Este repositório contém uma coleção de exercícios em C# focados em:

- Programação Orientada a Objetos (POO)
- Estruturas de Controle (`for`, `while`, `do-while`, `if/else`, `switch`)
- Manipulação de arquivos
- Interfaces e Polimorfismo
- Enums
- Tratamento de Exceções (Try/Catch)
- Conversão e operações matemáticas
